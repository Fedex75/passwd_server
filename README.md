# passwd_server
# Uso
Clonar el repositorio, instalar las dependencias e iniciarlo
```
git clone https://github.com/Fedex75/passwd_server.git
cd passwd_server
npm i
npm run start
```
# Configuracion
Especificar en el archivo .env:
- PORT
- MONGO_URI

(opcional; los valores por defecto funcionan correctamente)
